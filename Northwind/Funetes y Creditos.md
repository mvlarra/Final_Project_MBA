## 📚 Fuentes y Créditos

Esta aplicación fue desarrollada con fines educativos como parte de un análisis de Market Basket (MBA).

### 📦 Dataset utilizado:
**Northwind Sample Database** (versión SQLite)  
- Base de datos ficticia con pedidos, productos y categorías  
- Fuente: [Northwind SQLite en GitHub](https://github.com/jpwhite3/northwind-SQLite)  
- Licencia: Uso libre para fines educativos y no comerciales

### 🛠 Herramientas utilizadas:
- **Python 3.x**
- **pandas** – manipulación de datos
- **mlxtend** – generación de reglas de asociación (Apriori)
- **Streamlit** – interfaz interactiva
- **sqlite3 / pyodbc** – conexión a base de datos

> Todos los recursos utilizados tienen licencias open source y fueron empleados únicamente con fines de aprendizaje.
